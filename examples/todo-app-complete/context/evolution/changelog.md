# Changelog

All notable changes to the Todo Application project.

## [1.0.0] - 2024-01-20

### Added
- User authentication (signup, login, logout)
- Todo CRUD operations (create, read, update, delete)
- JWT-based authentication
- PostgreSQL database with Prisma ORM
- React frontend with TypeScript
- Express backend API
- Basic UI for todo management

### Fixed
- Todo persistence bug (missing database transaction commit)
- Authentication token storage (moved from localStorage to httpOnly cookies)

### Changed
- Updated authentication approach based on learnings
- Improved error handling in API

## [0.1.0] - 2024-01-15

### Added
- Initial project setup
- Context Mesh structure
- Project intent and feature intents
- Technical decisions documentation
