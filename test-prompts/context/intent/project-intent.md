# Project Intent: TaskFlow

## What
A simple task management application that allows users to create, organize, and track tasks with categories and due dates.

## Why
Help individuals and small teams stay organized and productive by providing a clean, intuitive interface for task management without the complexity of enterprise solutions.

## Acceptance Criteria
- Users can create, edit, and delete tasks
- Tasks can be organized by categories
- Tasks have due dates with reminders
- Clean, responsive UI

## Scope
- User authentication
- Task CRUD operations
- Category management
- Due date tracking
