# Feature: Task CRUD

## What
Allow users to create, read, update, and delete tasks.

## Why
Core functionality for task management - users need to manage their tasks.

## Acceptance Criteria
- Users can create tasks with title, description, due date
- Users can view all their tasks
- Users can edit existing tasks
- Users can delete tasks
- Tasks persist in database
