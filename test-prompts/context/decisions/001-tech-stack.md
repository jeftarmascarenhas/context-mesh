# Decision: Tech Stack

## Context
Starting TaskFlow web app project, choosing technologies.

## Decision
- Frontend: React + TypeScript + Tailwind CSS
- Backend: Node.js + Express
- Database: PostgreSQL
- Auth: JWT tokens

## Rationale
- React: Popular, great ecosystem, team experience
- TypeScript: Type safety, better DX
- Tailwind: Rapid UI development
- PostgreSQL: Reliable, scalable relational database
- JWT: Stateless auth, good for API

## Alternatives
- Vue.js - Less team experience
- MongoDB - Relational data fits better with PostgreSQL
- Session-based auth - More complex for API
